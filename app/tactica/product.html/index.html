플레이어가 바로 보는 정보

검색, 복기, 비교에 필요한 화면을 분리해 TFT 기록을 빠르게 읽을 수 있게 구성합니다.

1

Riot ID 전적 검색

Riot ID와 태그를 입력하면 TFT 랭크 정보와 최근 경기 흐름을 확인합니다.

2

매치 히스토리 분석

각 경기의 등수, 유닛, 특성, 아이템, 증강 정보를 정리해 복기합니다.

3

덱과 메타 탐색

덱 구성과 챔피언 정보를 비교해 다음 게임의 선택지를 좁힙니다.

4

상대 덱 비교

내 조합과 상대 조합의 전투 구도를 비교해 위험한 매치업을 표시합니다.

5

승률 추정 표시

알고리즘 기반의 추정값을 정보로 제공하며, 플레이 입력은 사용자가 직접 결정합니다.

6

한국어 TFT 데이터

챔피언, 특성, 아이템, 증강 정보를 한국어 기준으로 표시합니다.

사용 흐름

플레이어가 매번 필요한 정보를 적은 단계로 확인하도록 설계합니다.

Riot ID 검색 플레이어 프로필과 TFT 랭크 정보를 불러옵니다.
최근 경기 확인 최근 매치의 등수와 조합 흐름을 요약합니다.
덱 구성 비교 유닛, 특성, 아이템, 증강을 함께 확인합니다.
다음 판단 준비 정보를 참고해 배치, 전환, 아이템 선택을 직접 결정합니다.

데이터와 공정성

Tactica TFT는 플레이어가 직접 판단할 수 있도록 정보를 보여주는 도구입니다.

Riot API 기반

프로필, 랭크, 매치 기록, TFT 정적 데이터는 Riot API와 Data Dragon 데이터를 사용합니다.

자동 플레이 없음

게임 클라이언트를 제어하거나 입력을 전송하지 않으며, 봇·매크로·클라이언트 변조 기능을 제공하지 않습니다.
